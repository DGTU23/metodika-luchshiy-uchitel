# Скрипты для нагрузочного тестирования

## Требования
- Установленный [tcpkali](https://github.com/machinezone/tcpkali) (для TCP)
- (Опционально) wrk2 для HTTP, iperf3 для UDP

## Использование
1. Запустите ваш сервер на localhost:8080.
2. Выполните `bash load_test_tcpkali.sh`.
3. Результаты появятся в файлах results_*.txt.

## Настройка
Вы можете изменить параметры в скрипте: число соединений, частоту запросов, полезную нагрузку.
