# Инженерный практикум: методика для дисциплины «Системное программирование»

**Автор:** Исабекова Тамила Илахидиновна, зав. кафедрой ПМиИ, ДГТУ  
**Конкурс:** «Лучший учитель», номинация «Мост в профессиональное будущее»

Этот репозиторий содержит все материалы методики, описанной в презентации.  
Методика позволяет научить студентов не просто писать код, а **исследовать, моделировать и доказывать улучшения** — именно эти навыки ждут работодатели.

## 📂 Структура

- **`/templates`** — шаблоны документов, которые заполняют студенты в каждом проекте.
- **`/examples`** — примеры выполненных работ (на примере проекта TCP-сервера).
- **`/scripts`** — скрипты для автоматизации нагрузочного тестирования и сбора метрик.

## 🚀 Как использовать

1. Ознакомьтесь с 6-шаговым конвейером (описан в презентации).
2. Выдайте студентам шаблоны из папки `/templates`.
3. Поставьте задачу (например, разработать TCP-сервер с заданными требованиями).
4. После завершения проекта оцените работу по рубрике (`templates/Rubric_100.pdf`).
5. Примеры из `/examples` помогут студентам понять ожидаемый уровень.
