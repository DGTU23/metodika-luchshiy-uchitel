# Инженерный практикум: методика для вуза

Автор: Исабекова Тамила Илахидиновна, ДГТУ

Этот репозиторий содержит материалы методики, представленной на конкурс "Лучший учитель" в номинации "Мост в профессиональное будущее".

## 📂 Содержание

*   **/templates** — Шаблоны документов для студентов:
    *   `Model_Card.docx` — описание модели и её ограничений.
    *   `Experiment_Protocol.docx` — протокол проведения измерений.
    *   `Rubric.pdf` — критерии оценивания (100 баллов).
*   **/examples** — Примеры выполненных студентами работ:
    *   `Model_Card_Example_TCP.pdf` — пример паспорта системы для TCP-сервера.
    *   `Results_Table_Example.png` — таблица с доказательством улучшений.
*   **/scripts** — Скрипты для воспроизведения экспериментов (нагрузочное тестирование).

## 🚀 Как использовать

1.  Скопируйте шаблоны из папки `/templates`.
2.  Выдайте студентам задание в соответствии с 6-шаговым конвейером.
3.  Используйте рубрику для прозрачной оценки.
